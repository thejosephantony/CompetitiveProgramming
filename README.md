CompetitiveProgramming

Repositório dedicado ao estudo e à prática de Programação Competitiva, reunindo soluções de problemas computacionais, implementações de algoritmos clássicos e exercícios voltados ao desenvolvimento do raciocínio lógico, da capacidade analítica e da escrita de código eficiente.

📌 Fundamentação

A Programação Competitiva é uma área da Ciência da Computação centrada na resolução de problemas algorítmicos sob restrições de tempo e memória. Seu propósito não é apenas encontrar respostas funcionais, mas construir soluções corretas, eficientes e fundamentadas em princípios matemáticos e computacionais.

Essa prática envolve conceitos como:

- análise de complexidade assintótica
- estruturas de dados
- paradigmas de algoritmos
- matemática discreta
- teoria dos grafos
- otimização computacional
- modelagem formal de problemas

Além do aspecto técnico, contribui para o desenvolvimento da abstração, disciplina intelectual e tomada de decisão sob pressão.

🎯 Objetivos do Repositório

Este repositório tem como finalidade:

- registrar soluções desenvolvidas ao longo dos estudos
- consolidar conhecimentos teóricos e práticos
- servir como base de revisão futura
- acompanhar evolução técnica contínua
- auxiliar na preparação para competições e entrevistas
- fortalecer domínio da linguagem C++

💻 Linguagem Utilizada

- C++, amplamente adotada em programação competitiva devido ao desempenho, controle eficiente de recursos e utilização da STL (Standard Template Library).

🧠 Conteúdos Frequentes

Os problemas resolvidos neste repositório abrangem tópicos clássicos e avançados da Ciência da Computação, entre eles:

Estruturas de Dados

Estudo de formas eficientes de organizar e manipular informações em memória:

- Vetores e Matrizes
- Pilhas (Stacks)
- Filas (Queues)
- Deques
- Listas Encadeadas
- Conjuntos e Mapas Balanceados
- Hash Tables
- Heaps / Priority Queues
- Union-Find (Disjoint Set Union)
- Árvores e Fenwick Tree / Segment Tree

Algoritmos Fundamentais

Técnicas centrais para processamento e busca eficiente:

- Ordenação (Sorts clássicos e STL)
- Busca Linear e Busca Binária
- Two Pointers
- Sliding Window
- Prefix Sum / Difference Array
- Técnicas de varredura (Sweep Line)

Paradigmas de Resolução

Modelos clássicos de construção de algoritmos:

- Divisão e Conquista
- Programação Dinâmica
- Algoritmos Gulosos (Greedy)
- Backtracking
- Recursão
- Branch and Bound

Grafos

Modelagem de relações entre entidades e caminhos:

- BFS e DFS
- Caminho mínimo (Dijkstra, Bellman-Ford)
- Floyd-Warshall
- Árvore Geradora Mínima
- Ordenação Topológica
- Componentes Conexas
- Fluxo em Redes

Matemática Computacional

Base matemática aplicada à resolução de problemas:

- Aritmética Modular
- Máximo Divisor Comum e MMC
- Números Primos e Crivo
- Exponenciação Rápida
- Combinatória
- Probabilidade
- Teoria dos Números
- Álgebra básica aplicada

Strings e Processamento de Texto

Técnicas voltadas à manipulação eficiente de sequências:

- Busca de padrões
- Hashing de Strings
- KMP
- Trie
- Prefix Function / Z-Function

Técnicas Avançadas

Conteúdos recorrentes em problemas de maior dificuldade:

- Bitmask
- Meet in the Middle
- Sparse Table
- Binary Lifting
- Coordinate Compression
- Offline Queries
